# softly-replace

選択された範囲のテキストを「柔らかく言い換える」Chrome拡張です。

とりあえず手元で動いたという程度です。

使ってみたい方はリポジトリをcloneした上で、Chromeの「パッケージ化されていない拡張機能を読み込む」からディレクトリごと読み込んでください。

OpenAIのいわゆるChatGPT APIを使っています。

拡張機能の設定ページからOpenAIのAPIキーを設定してください。APIキーはサーバに保存されたりしないので安心してください。
